# Espanso Search

Search espanso.org matches with `esp` command.

## Installation

Requires Espanso and [node](https://nodejs.org/).

#### Download

```
git clone git@github.com:davidsword/espanso-search.git
cd espanso-search/
```

#### Install globally

```
npm install -g .
```

## Usage 

```
$ esp <keyword> 
```
